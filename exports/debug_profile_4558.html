    Oiê! Cheguei com um estudo simples, mas com bons insights. Fiz uma breve análise de mercado com dados públicos de crédito a veículos, usando Microsoft SQL Server e Power BI.

    Tem um resultado final conciso, mas há várias camadas de análise! Vamos a 3 pontos principais:

    1️⃣ Temos várias empresas envolvidas - algumas B2B e outras B2C nessa cadeia produtiva. Todos os pontos importam para a análise, principalmente a viabilidade de compra do consumidor final (um veículo adquirido com crédito de terceiros).

    2️⃣ São usados dados de mercado, pois era algo que a empresa não estava olhando. Dentro dos seus processos a empresa tinha pleno domínio e muitos dados disponíveis, mas ainda faltava algo para apoiar o decisor.

    3️⃣ Era importante olhar para fora e entender não só o cenário macro, mas também os dados históricos de um período anterior até chegar no presente. Assim, seria possível ter um panorama mercado-empresa na perspectiva passado-presente, de modo a permitir insights para o futuro.
